Voting Bot
==========

To run this bot:

1. Put `qu` binary into the working directory of the bot.
2. Create a `settings.json` file in the working directory with the content:

    {
        "token": "<BOT_TOKEN>",
        "chatId": <CHAT_ID>,
        "neuronId": <NEURON_ID>,
        "pemFile": "<PATH_TO_PEM>"
    }
